## 1. Điểm mạnh của Numpy so với Python là có thể mô phỏng được mảng nhiều chiều 
Numpy có thể tạo ra 1 vector:
```python
import numpy as np
vector = np.array([1,2,3]) 
print(vector) 
print(type(vector))
print(vector.shape)

-> [1,2,3]
   <class 'numpy.ndarray'>
   (3,)
```
